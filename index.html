A Real-world Display Inverse Rendering Dataset

TL;DR

We present a novel real-world display inverse rendering dataset, DIR. The DIR dataset consists of:

  • 144 images of 16 objects captured under OLAT lighting
  • Ground truth surface normal map
  • Polarization RAW signals
  • Stereo pair images
  • Carefully-calibrated light informations

We provide the dataset in an organized format, along with benchmark results for photometric stereo, single-view & multi-light inverse rendering, and baseline code for display inverse rendering. Please refer to our GitHub page for details.

Gallery

Overview

The capture system consists of an LCD monitor as the light source and a stereo polarized camera for imaging. For each object, we sequentially projected 144 (16 × 9) OLAT patterns and captured them. The scanned mesh was then aligned to the scene pose to obtain ground truth depth and normal maps.
